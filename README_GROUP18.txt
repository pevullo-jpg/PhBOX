GRUPPO FUNZIONALE 18

Obiettivo:
- nessuna modifica codice per cambiare account Google
- configurazione OAuth Web Client ID da interfaccia
- login / logout / cambio account da pagina Impostazioni

IMPORTANTE
L'account Google si cambia SOLO da interfaccia.
Il Web Client ID si salva da interfaccia.

UNICA OPERAZIONE ESTERNA OBBLIGATORIA
Su Google Cloud Console devi configurare una sola volta:
- OAuth Client ID di tipo Web application
- Authorized JavaScript origins del dominio del sito

Esempio GitHub Pages:
https://TUOUSERNAME.github.io
